## Installation Notes
  * You need to have a Systrans API Key.
  * The Systrans Translation API library must be downloaded into your libraries folder. It's available at:
    https://github.com/SYSTRAN/translation-api-php-client
    Proper libraries structure:

    - libraries/
      - Systrans/
        - lib/
        - tests/
        - apiKey.txt
        - autoload.php
        - composer.json
        - composer.phar
        - LICENSE.txt
        - README.md
